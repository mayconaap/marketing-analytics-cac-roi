# 📊 Projeto 4: Análise de Marketing - CAC, ROI e Retenção

---

## 📋 Descrição

Projeto de análise de marketing para calcular CAC (Custo de Aquisição), ROI (Retorno sobre Investimento) e comparar com custo de retenção de clientes.

## 🗺️ Roadmap

- [ ] Parte 1: Configuração do ambiente e GitHub
- [ ] Parte 2: Análise exploratória dos dados
- [ ] Parte 3: Cálculo de CAC por canal
- [ ] Parte 4: Análise de ROI por campanha
- [ ] Parte 5: Comparação CAC vs CRC (custo de retenção)
- [ ] Parte 6: Insights e recomendações
